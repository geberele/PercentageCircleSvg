#Percentage Circle Svg
This PHP class allows you to get Percentage Circle Svg. Really customizable, you can change the percentage, the colors, the text, etc...

##How to use it
```
$svg = new PercentageCircleSvg(50, 25);
print $svg->getSvg();
```
